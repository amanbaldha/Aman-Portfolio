# Aman's Portfolio 🚀

A modern, responsive, cinematic personal portfolio website showcasing my experience and projects as an Android Developer.

### 🌟 Features
- **Cinematic Dark Theme**: Custom UI with smooth scroll reveals and dynamic hover states.
- **Backend-Free Contact Form**: Fully functional email routing via EmailJS.
- **Specialization Showcase**: Highlights native Android, KMP, and on-device AI expertise.

### 🛠 Tech Stack
- HTML5, Vanilla CSS3, JavaScript
- EmailJS (Serverless contact form)

🔗 **Live Website**: https://amanbaldha.github.io/Aman-Portfolio/
